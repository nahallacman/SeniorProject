Creates a layer.

This function is used to create an additional layer that can be used for PIP, Cursor or other porpouse.

Parameters
LayerCorresponds with layer. This value can be either GFX_PIP_LAYER or GFX_CURSOR_LAYER.
ConfigThis value makes configuration for the Layer that can be combained with the following flags: GFX_LAYER_TRANS_EN GFX_LAYER_OUTPUT_NORMAL GFX_LAYER_OUTPUT_XOR GFX_LAYER_OUTPUT_XORNOT GFX_LAYER_OUTPUT_NOT and alpha color 4 bit value. For instance, Config can be ( GFX_LAYER_TRANS_EN | GFX_LAYER_OUTPUT_NORMAL | 0xA ) where 0xA is value of transparency.
XStartThis value configures the X start position for the Layer
YStartThis value configures the Y start position for the Layer
WidthThis value configures the width for the Layer
HeightThis value configures the height for the Layer
