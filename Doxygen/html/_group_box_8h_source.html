208 /*********************************************************************
209 * Function: WORD GbDraw(void *pObj)
210 *
211 * Overview: This function renders the object on the screen using
212 * the current parameter settings. Location of the object
213 * is determined by the left, top, right and bottom parameters.
214 * The colors used are dependent on the state of the object.
215 * The font used is determined by the style scheme set.
216 *
217 * When rendering objects of the same type, each object must
218 * be rendered completely before the rendering of the next object
219 * is started. This is to avoid incomplete object rendering.
220 *
221 * PreCondition: Object must be created before this function is called.
222 *
223 * Input: pGb - Pointer to the object to be rendered.
224 *
225 * Output: Returns the status of the drawing
226 * - 1 - If the rendering was completed and
227 * - 0 - If the rendering is not yet finished.
228 * Next call to the function will resume the
229 * rendering on the pending drawing state.
230 *
231 * Example:
232 * See GbCreate() example.
233 *
234 * Side Effects: none
235 *
236 ********************************************************************/
237 WORD GbDraw(void *pObj);
