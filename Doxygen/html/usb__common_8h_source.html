451 /*******************************************************************************
452  Function:
453  BOOL <Event-handling Function Name> ( USB_EVENT event,
454  void *data, unsigned int size )
455 
456  Description:
457  This routine is a "call out" routine that must be implemented by
458  any layer of the USB SW Stack (except the HAL which is at the
459  root of the event-call tree that needs to receive events. When
460  an event occurs, the HAL calls the next higher layer in the
461  stack to handle the event. Each layer either handles the event
462  or calls the layer above it to handle the event. Events are
463  identified by the "event" parameter and may have associated
464  data. If the higher layer was able to handle the event, it
465  should return TRUE. If not, it should return FALSE.
466 
467  Preconditions:
468  USBInitialize must have been called to initialize the USB SW
469  Stack.
470 
471  Paramters:
472  USB_EVENT event - Identifies the bus event that occured
473  void *data - Pointer to event-specific data
474  unsigned int size - Size of the event-specific data
475 
476  Return Values:
477  None
478 
479  Remarks:
480  The function is name is defined by the layer that implements
481  it. A pointer to the function will be placed by into a table
482  that the lower-layer will use to call it. This requires the
483  function to use a specific call "signature" (return data type
484  and values and data parameter types and values).
485
