240 * Function: WORD DmDraw(void *pObj)
241 *
242 * Overview: This function renders the object on the screen using
243 * the current parameter settings. Location of the object
244 * is determined by the left, top, right and bottom
245 * parameters. The colors used are dependent on the state
246 * of the object. The font used is determined by the style
247 * scheme set.
248 *
249 * When rendering objects of the same type, each object must
250 * be rendered completely before the rendering of the next
251 * object is started. This is to avoid incomplete object rendering.
252 *
253 * PreCondition: Object must be created before this function is called.
254 *
255 * Input: pDm - Pointer to the object to be rendered.
256 *
257 * Output: Returns the status of the drawing
258 * - 1 - If the rendering was completed and
259 * - 0 - If the rendering is not yet finished.
260 * Next call to the function will resume the
261 * rendering on the pending drawing state.
262 *
263 * Example:
264 * See DmCreate() Example.
265 *
266 * Side Effects: none
267 *
268 ********************************************************************/
