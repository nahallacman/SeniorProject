57  100*(1 - 1/DEBUG_OPTIMAL_CHARGE_DIVIDER)% of AVdd. For example:

116 typedef struct tag_DEBUGDELAY {
117 INT16 error; // sample error value for the adjusted delay.
118  // It should be as low as possible. If the mistake is
119  // bigger than 128 for 10-bit ADC or bigger than
120  // 512 for 12-bit ADC then the CTMU current must be
121  // adjusted first.
