176 /*****************************************************************************
177  Function:
178  UDP_SOCKET UDPOpen(UDP_PORT localPort, NODE_INFO* remoteNode,
179  UDP_PORT remotePort)
180 
181  Summary:
182  Macro of the legacy version of UDPOpen.
183 
184  Description:
185  UDPOpen is a macro replacement of the legacy implementation of UDPOpen.
186  Creates a UDP socket handle for transmiting or receiving UDP packets.
187  Call this function to obtain a handle required by other UDP function.
188 
189  Precondition:
190  UDPInit() must have been previously called.
191 
192  Parameters:
193  localPort - UDP port number to listen on. If 0, stack will dynamically
194  assign a unique port number to use.
195  remoteNode - Pointer to remote node info (MAC and IP address) for this
196  connection. If this is a server socket (receives the first packet)
197  or the destination is the broadcast address, then this parameter
198  should be NULL.
199  remotePort - For client sockets, the remote port number.
200 
201  Return Values:
202  Success -
203  A UDP socket handle that can be used for subsequent UDP API calls.
204  Failure -
205  INVALID_UDP_SOCKET. This function fails when no more UDP socket
206  handles are available. Increase MAX_UDP_SOCKETS to make more sockets
207  available.
208 
209  Remarks:
210  When finished using the UDP socket handle, call the UDPClose() function
211  to free the socket and delete the handle.
212  ***************************************************************************/
213 #define UDPOpen(localPort,remoteNode,remotePort) UDPOpenEx((DWORD)remoteNode,UDP_OPEN_NODE_INFO,localPort,remotePort)
