296  #define MTOUCH_RELEASE_THRESH_FACTOR 0.5 // <-- Defines the hysteresis between press and release
297  // In other words, '0.4' would result in a release
298  // threshold equal to 40% of the press threshold.
299  //
300  // Example: With a finger on the sensor, the reading
301  // settles to a value (1000). When a finger is added,
302  // we see the reading increase (1200). The press
303  // threshold is then set to determine how much shift
304  // we want to see before we get a press. In this case
305  // we see a shift of 200. So we can set the threshold
306  // to 160. The release threshold is a percentage of
307  // the press threshold. If set to 0.4, it will be
308  // 40% of the 160 press threshold, or '64' counts.
309  // So the sensor will remain pressed until the reading
310  // returns to a value of 1064 or less.
311  //
312  // Allowable Range :: [ 0.30 to 1.00 ]
