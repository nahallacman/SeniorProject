362  - FS_APPEND - Append data to an existing file
363  - FS_WRITEPLUS - Create a new file or replace an existing file (reads also enabled)

382  end of the file using the FSfseek function. If the file was not
383  found by FILEfind, a new file will be created if the mode was specified as
384  a write or append mode. In these cases, a pointer to the heap or
385  static FSFILE object array will be returned. If the file was not
386  found and the mode was specified as a read mode, the memory
387  allocated to the file will be freed and the NULL pointer value
