439  // Outputs the next byte of the 'read stream' packet. The 'stream' is an array of vectors
440  // (a pointer + a length). To get the next byte, the iterator function accesses the pointer
441  // within the current vector and grabs the Nth value until N is incremented to equal the
442  // length value of the vector. When this happens, the current vector is incremented. It
443  // continues until the vector it is pointing to has a length of 0. When this happens, the
444  // hasNext flag is cleared to stop the iteration.

446  // Call Number: Outputs:
447  // 1 Packet Length (including mComm's automatic checksum)
